# Network Labs

Hands-on networking labs built with GNS3 to demonstrate practical skills in Cisco networking, troubleshooting, and network administration.

## About This Repository

This repository contains my networking lab projects created in GNS3. Each lab focuses on a specific networking technology and includes:

* GNS3 project files
* Device configurations
* Topology diagrams
* Verification and testing results
* Troubleshooting notes
* Screenshots

## Lab Categories

### First Hop Redundancy Protocols (FHRP)

* HSRP (Hot Standby Router Protocol)
